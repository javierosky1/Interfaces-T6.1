# T6.1 (anteriormente T4.1)

## Descripción

Es una aplicación de gestión de pedidos destinada a un bar. Permite visualizar mesas, el número de productos pedidos y el importe. También se pueden crear nuevos pedidos, añadirles productos y ver el resumen del nuevo pedido.

## Tecnologías

* Flutter
* Dart

## Características

* Estructura Model-View-ViewModel (MVVM)
